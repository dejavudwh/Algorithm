### 解题思路

> 数组遍历 找规律 记忆值

和118题一样，但是为了节省时间可以保存上一层的结果，而不需要保存所有的值

### 代码

```java
class Solution {
    public List<Integer> getRow(int rowIndex) {
        List<Integer> result = new ArrayList<>();
        result.add(1);
        int pre = 1;
        for (int i = 1; i <= rowIndex; i++) {
            for (int j = 1; j < i; j++) {
                int temp = result.get(j);
                result.set(j, pre + result.get(j));
                pre = temp;
            }
            result.add(1);
        }

        return result;
    }
}
```
